MySQL status 详解

发表于 Apr 22 2018 | 分类于 mysql

MySQL 提供了一些列的状态用来帮助我们诊断服务状况，有些状态非常有用，可以帮助我们快速定位服务异常，比如最大连接数，当前连接数等等，下表详细列举了所有的状态以及他们的详细解释，具体见官方文档：

https://dev.mysql.com/doc/refman/8.0/en/server-status-variables.html
